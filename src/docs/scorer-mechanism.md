# 场景测试是怎么打分的？

场景测试的每个题目都会根据模型回答独立评分（百分制）。而**评分器**字段将决定评分的方式。
- **不填评分器**：按填写的关键词估分，最高约90分。
- **填评分器**：依照填写的答案判分，可到100分。

## 不填评分器

- 基础分值为60
- 回答达到最小字数＋15分
- 每命中一个关键词＋4分，最高＋15分

想提高评分就**多写几个常出现的关键词（或是重复的关键词）**（≥4 个 → 可到 90）。

## 填评分器

问题能得出明确答案时用（注：答案写入“期望答案”项，而非“关键词”项）。

| 评分器                    | 适合           |
|------------------------|--------------|
| `exact`（最常用）           | 一段确定的字符      |
| `set`                  | 不分先后的一组字符    |
| `table` / `structured` | 表格 / 规定格式数据  |
| `needle`               | 长文里找指定的一句话   |
| `ifeval` / `bfcl`      | 按多条指令 / 调用工具 |

## 得分标准

- 没答出 → 0 分
- 输出被截断 → 0分，但分数不计入总成绩
- **70 分**＝通过
- 
- **90 分**＝自动给模型贴对应能力标签（需在「设置」开启）
